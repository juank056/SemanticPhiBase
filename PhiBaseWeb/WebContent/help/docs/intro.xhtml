1. ¿Por qué manual de Usuario en línea?

En este manual de usuario se encuentran descritas todas las funcionalidades que dispone el Sistema Semantic PHI-BASE Web Interface. Se ha realizado este manual a manera de páginas web de tal forma que siempre esté disponible para los usuarios en el momento en que lo necesiten.

2. Navegación:

Para consultar los diferentes servicios ofrecidos por el sistema, se cuenta con un panel en la parte izquierda de la pantalla. En este panel están descritos los Servicios Básicos (para todos los usuarios), junto con esta introducción. Además de esto, se incluyen los servicios a los que un usuario autenticado tiene autorización de acceso. El panel es como se muestra a continuación:

Desde este panel se puede acceder a la documentación de cada uno de los servicios haciendo click sobre alguno de ellos. Es importante notar que esta imagen puede variar dependiendo de los servicios a los que el usuario tenga acceso o de si se desarrollan nuevas funcionalidades en el sistema en un futuro.
