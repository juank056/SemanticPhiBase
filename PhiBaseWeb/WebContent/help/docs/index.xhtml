Objetivo Esta pantalla, es la pantalla inicial de acceso al sistema Semantic PHI-BASE Web Interface. Desde aqui los usuarios podrán autenticarse utilizando las credenciales que se les hayan proporcionado para tal fin.
Consideraciones Es importante que los usuarios hayan recibido un correo electrónico inicial en donde se indican sus credenciales para su primer ingreso. En caso de que no se hayan recibido las credenciales se debe contactar con el administrador del sistema para que proceda a reenviarlas.
Pantalla Principal
Observaciones Para ingresar al sistema se debe introducir el nombre de usuario (Un E-Mail que identifica al usuario dentro del sistema), junto con su contraseña de acceso. En el momento en el que el usuario ingresa los datos y da click sobre el bóton "Ingresar", ocurren las siguientes acciones:
   1. Se valida que el usuario exista en el sistema y que su contraseña sea correcta.
   2. Se valida que el usuario tenga un estado que le permita ingresar al sistema (Activo).
   3. Se valida la caducidad de su contraseña y la fecha de último cambio para saber si debe cambiarla.

En caso de que todo sea correcto, se procederá a cargar la pantalla de menú principal de la aplicación (Ver: Menú Principal), en donde se desplegarán todos los servicios a los que el usuario tenga acceso.
En caso de que el usuario tenga que cambiar su contraseña, se navegará hacia la pantalla de cambio de contraseña (Ver: Actualizar Contraseña).
Finalmente, si el nombre de usuario o la contraseña son incorrectos, se quedará en la misma pantalla indicando el mensaje de error. Es importante saber que se tienen un número limitado de intentos fallidos (Parámetro del Sistema), y en el momento en que se cumplan, se procederá a inactivar el usuario por un tiempo parametrizado o hasta que sea activado de nuevo manualmente.
