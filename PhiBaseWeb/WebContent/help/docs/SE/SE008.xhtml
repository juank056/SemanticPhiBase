Objetivo Esta pantalla permite consultar las queries que han sido ejecutadas por los diferentes usuarios del sistema hacia el Sistema Semantic PHI-BASE.
Consideraciones Ninguna.
Pantalla Principal
Parámetros Recibidos Ninguno.
Buscador 1. Por rango de fechas de ejecución.
2. Por Usuario del Sistema.
3. Por rango de fechas y Usuario del Sistema.
Acciones 1. Detalle Query Semantic PHI BASE
2. Cargar Parámetros de Query
3. Ejecutar Query y Ver Resultados
Observaciones Ninguna.
Detalle Query Semantic PHI BASE
Campo Descripción Tipo de Campo Consideraciones
Usuario Usuario que ha ejecutado la query. OUTPUT 1. No es posible modificar su valor.
Tiempo de Ejecución Fecha y hora de ejecución de la query. OUTPUT 1. No es posible modificar su valor.
Sparql Query Texto de la query ejecutada en la base de conocimiento de Virtuoso. OUTPUT 1. No es posible modificar su valor.
Descripción Query Texto ingresado por el usuario para describir la Query que ha sido ejecutada en el Sistema. OUTPUT 1. No es posible modificar su valor.
Observaciones Ninguna.
Cargar Parámetros de Query
Observaciones 1. Esta opción navegará hacia la pantalla de Consulta Semantic PHI-BASE y cargará en el tab de filtro de selección de búsqueda todos los parámetros de ejecución que han sido seleccionados por el usuario que ejecutó la query. Ver Consulta Semantic PHI-BASE.
Ejecutar Query y Ver Resultados
Observaciones 1. Esta opción navegará hacia la pantalla de Consulta Semantic PHI-BASE y ejecutará la query nuevamente y cargará los resultados obtenidos de acuerdo a los datos que se encuentren disponibles en la base de conocimiento de Semantic PHI-BASE. Ver Consulta Semantic PHI-BASE.
