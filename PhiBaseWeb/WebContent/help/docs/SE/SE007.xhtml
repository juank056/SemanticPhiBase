Objetivo Esta pantalla permite realizar la consulta de los bloqueos que se han realizado a un usuario por intentos fallidos de Login dentro del sistema.
Consideraciones Ninguna.
Pantalla Principal
Parámetros Recibidos Ninguno.
Buscador 1. Por rango de fechas de bloqueo.
2. Por Usuario del Sistema.
3. Por rango de fechas y Usuario del Sistema.
Acciones Ninguna.
Observaciones 1. Los campos que se consultan de cada Log de ingreso son los siguientes:
   a. Usuario Bloqueado: Se trata del identificador del usuario que ha sido bloqueado en el sistema.
   b. Tiempo de Bloqueo: Fecha y hora de generación del bloqueo.
   c. Intentos Fallidos: Número de intentos fallidos de Login que ha realizado el usuario para ingresar al sistema.
   d. Tiempo Desbloqueo: Fecha y Hora en que se ha realizado el desbloqueo del usuario para permitir su ingreso.
   e. Usuario Desbloqueo: Usuario que ha realizado el desbloqueo del usuario bloqueado. Puede ser "Automático Sistema" en caso de que haya sido un desbloqueo automático.
   f. Desbloqueo Automático: Indicador para saber si se trata de un desbloqueo automático o de si ha sido un usuario el que ha realizado el desbloqueo manualmente.
